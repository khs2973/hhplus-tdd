### **이번주 KPT 회고**

### Keep
- 기능 구현 전에 테스트를 먼저 작성할것
- 예외 케이스들을 테스트로 먼저 생각할것

### Problem
- 테스트 코드를 작성하는게 처음이여서 어색하다보니 기능을 먼저 구현하고 테스트 코드를 작성
   - 테스트 코드를 많이 연습하며 실력을 늘릴것

### Try
- 테스트 코드를 더 이해하고 다양한 정책이나 예외 케이스를 검증해볼것
